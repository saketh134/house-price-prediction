# house-price-prediction
Project Description:

This project aims to predict house prices using Data Science and Machine Learning techniques. By analyzing various property features such as area, number of bedrooms, bathrooms, parking availability, and furnishing status, the model estimates the expected selling price of a house. The project demonstrates the complete machine learning workflow from data preprocessing to model evaluation.

Objective:

To build a predictive model for estimating house prices.
To identify the key factors that influence housing prices.
To apply data preprocessing, exploratory data analysis, and machine learning techniques for accurate predictions.

Technologies Used:

Python
Pandas
NumPy
Matplotlib
Seaborn
Scikit-learn
Jupyter Notebook

Process Applied:

1.Collected and loaded the housing dataset.
2.Performed data cleaning and preprocessing.
3.Conducted Exploratory Data Analysis (EDA) to understand patterns and relationships.
4.Encoded categorical features and prepared the data for modeling.
5.Split the dataset into training and testing sets.
6.Trained machine learning models such as Linear Regression and Random Forest Regressor.
7.Evaluated model performance using metrics like MAE, RMSE, and R² Score.
8.Generated house price predictions based on input features.

Outcome

The developed model successfully predicts house prices and helps understand the impact of different housing attributes on property value.
